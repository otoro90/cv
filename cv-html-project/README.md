# Proyecto CV HTML

Este proyecto es una hoja de vida (currículum vitae) creada utilizando HTML, CSS y JavaScript. Está diseñado para ser accesible, responsivo y fácil de personalizar.

## Estructura del Proyecto

El proyecto contiene los siguientes archivos:

- **index.html**: Este archivo es la estructura principal de la hoja de vida. Contiene secciones como encabezado, experiencia laboral, educación, habilidades y contacto. Se asegura de seguir buenas prácticas de accesibilidad, utilizando etiquetas semánticas y atributos ARIA donde sea necesario.

- **css/styles.css**: Este archivo contiene los estilos para la hoja de vida. Incluye un esquema de colores agradable y un diseño responsivo. Se aplican estilos para mejorar la legibilidad y la experiencia del usuario, como el uso de fuentes adecuadas, márgenes y espaciado.

- **js/main.js**: Este archivo puede incluir scripts para mejorar la interactividad de la hoja de vida, como un botón para imprimir la página en formato PDF. Se asegura de que el código sea accesible y no interfiera con la navegación del usuario.

## Instrucciones para Ejecutar el Proyecto

1. Clona este repositorio en tu máquina local.
2. Abre el archivo `index.html` en tu navegador web.
3. Personaliza el contenido de la hoja de vida editando el archivo `index.html`.
4. Modifica los estilos en `css/styles.css` según tus preferencias.
5. Si deseas agregar interactividad, edita el archivo `js/main.js`.

## Tecnologías Utilizadas

- HTML5
- CSS3
- JavaScript

## Personalización

Puedes personalizar la hoja de vida cambiando el contenido en `index.html`, ajustando los estilos en `css/styles.css` y añadiendo funcionalidades en `js/main.js`. Asegúrate de mantener las buenas prácticas de accesibilidad y diseño responsivo.

## Contribuciones

Las contribuciones son bienvenidas. Si deseas mejorar este proyecto, siéntete libre de hacer un fork y enviar un pull request.

## Impresión en PDF

Para imprimir la hoja de vida en formato PDF desde el navegador:

1. Abre `index.html` en tu navegador.
2. Haz clic en el botón "Imprimir CV" en la parte inferior de la página.
3. Selecciona "Guardar como PDF" en las opciones de impresión del navegador.
4. Ajusta la configuración de impresión si es necesario (por ejemplo, márgenes, orientación).
